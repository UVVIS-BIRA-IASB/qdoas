QDOAS Tools

QDOAS is delivered with four off-line modules :

convolution
a convolution/filtering tool;
ring
a tool to create ring effect cross-sections
usamp
a tool to create undersampling cross-sections
doas_cl
a command line tool for batch processing

The three first ones were already in WinDOAS.  If they can still be called from the QDOAS user interface, they are separate executables with their own user interface, menu options and XML configuration files.
